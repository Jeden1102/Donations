# Donations

Simple donations webpage.

# Technologies used

- Nuxt3
- Taiwlind
- Supabase
- Typescript
- Stripe
- axios

## Installation

1. Build and run dev.

```bash
npm run install
npm run dev
```
2. Copy env.example to .env and insert supabase connection and stripe configuration.

```bash
cp .env.example .env
```
